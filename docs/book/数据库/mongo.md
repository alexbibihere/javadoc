# MongoDb

## 基于分布式文件存储的数据库
- 非关系型数据库,以文档形式存储数据

- 支持的数据结构:  bson(binary json)

- 与mysql的区别

mysql用table 表
mongo  用collection 集合

## 应用场景

- 数据量大  \  读写都很频繁\   价值较低的数据,对事务性要求不高
